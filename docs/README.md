## Documentation Folder

This folder stores non-runtime project notes and checklists moved from the project root to keep the application directory clean.

- `UAT_TEST_CHECKLIST.md`
- `QA_TESTING_CHECKLIST.md`
- `PRE_DEPLOY_CHECKLIST.md`
- `Workflow Plan and Rules.md`
- `DISABLED_FEATURES_REMINDER.txt`

`docs/samples/` contains sample or reference-only layouts that are not part of the runtime entry flow.
